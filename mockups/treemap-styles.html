Treemap Visual Directions

Four approaches to elevate the block visualization. Click to expand any mockup.

A. Soft Depth

Subtle gradients top-to-bottom on each block, rounded corners, drop shadows between layers. Feels like physical tiles stacked on a surface. Warm, approachable.

Low effort

B. Frosted Glass

Semi-transparent blocks with frosted edges and soft glows. Inner content blocks have a translucent quality. Labels float above with blur backgrounds. Modern, premium.

Medium effort

C. Outlined Minimal

Thin luminous borders, dark fills with subtle color tint. Clean typography. Monochrome base with color accents on hover. Professional, data-viz feel.

Low effort

D. Neon Cartography

Dark background, vibrant saturated fills, glowing borders. Directory labels in bold capsules. File blocks have a subtle noise texture. Bold, distinctive, very non-DaisyDisk.

Medium effort

Implementation Notes

All four styles are achievable in SwiftUI Canvas with zero external dependencies. Options A and C are quick wins using gradient fills and stroke tweaks. Options B and D require a bit more drawing logic (glow effects via multiple strokes, noise textures via tiled patterns). Any can be combined — e.g., Soft Depth blocks with Neon borders.
