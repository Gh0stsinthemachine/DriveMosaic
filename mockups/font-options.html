Font Pairings on Frosted Glass

Same frosted glass style, four different type treatments. All fonts ship with macOS — zero dependencies.

1. SF Pro Rounded + SF Mono

Rounded sans for folder names — warm, friendly, distinctive. Monospace for sizes — technical precision. Good contrast between the two roles.

Font Availability on macOS

SF Pro Rounded — via .systemFont(.rounded) design in SwiftUI. SF Pro Display — the default system font at larger sizes. New York — via .serif design in SwiftUI, ships on macOS 11+. SF Mono — via .system(.monospaced). All zero-cost, zero-bundle, rendered natively by Core Text.
